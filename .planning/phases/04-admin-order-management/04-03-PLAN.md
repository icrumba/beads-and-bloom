  colors: jsonb().$type<string[]>().notNull().default([]),
  customizable: boolean().notNull().default(false),
  availability: availabilityEnum().notNull().default("ready_to_ship"),
  inStock: boolean().notNull().default(true),
  featured: boolean().notNull().default(false),
  materials: text(),
  careInfo: text(),
  sortOrder: integer().notNull().default(0),
  createdAt: timestamp({ withTimezone: true }).notNull().defaultNow(),

});

    **Step 1: Create product CRUD Server Actions** (per D-10, D-11, D-13, ADMIN-03)
    Create `src/actions/products.ts` with `"use server"` directive:

    Import `z` from `"zod/v4"` (per existing pattern in stripe.ts).

    Define `productSchema` with zod:
    ```
    name: z.string().min(1, "Name is required").max(255),
    description: z.string().min(1, "Description is required"),
    price: z.string().regex(/^\d+\.\d{2}$/, "Price must be in format X.XX"),
    category: z.string().min(1, "Category is required"),
    images: z.array(z.string()).default([]),
    colors: z.array(z.string()).default([]),
    customizable: z.boolean().default(false),
    availability: z.enum(["ready_to_ship", "made_to_order"]).default("ready_to_ship"),
    inStock: z.boolean().default(true),
    featured: z.boolean().default(false),
    materials: z.string().optional(),
    careInfo: z.string().optional(),
    sortOrder: z.number().int().default(0),
    ```

    Helper function `generateSlug(name: string)`: lowercase, replace non-alphanumeric with hyphens, collapse multiple hyphens, trim hyphens from ends. Per D-11 (slug auto-generated from name).

    `createProduct(formData: FormData)`:
    1. Extract all fields from formData. Parse images and colors from JSON strings (they arrive as stringified arrays from the client form).
    2. Validate with productSchema
    3. Generate slug from name
    4. Insert into products table: `db.insert(products).values({ ...parsed, slug }).returning()`
    5. `revalidatePath("/admin/products")`
    6. `redirect("/admin/products")`

    `updateProduct(id: number, formData: FormData)`:
    1. Same extraction and validation as create
    2. Generate new slug from name
    3. `db.update(products).set({ ...parsed, slug, updatedAt: new Date() }).where(eq(products.id, id))`
    4. `revalidatePath("/admin/products")`
    5. `revalidatePath("/admin/products/" + id + "/edit")`
    6. Also `revalidatePath("/")` and `revalidatePath("/products")` to update storefront
    7. `redirect("/admin/products")`

    `deleteProduct(id: number)`:
    1. `db.delete(products).where(eq(products.id, id))`
    2. `revalidatePath("/admin/products")`
    3. Also `revalidatePath("/")` and `revalidatePath("/products")` to update storefront
    4. Return `{ success: true }`
