### Decisions
- Voice should impress adults (primary buyers 25-45), not read as teen-to-teen energy
- Always weave in signature details: ocean charms, custom colors, handwritten thank-you notes
- All skills kept (16 total), skill selection complete
- Stack: Next.js 15 + Neon Postgres + Drizzle ORM + Stripe + Cloudinary + Vercel Pro
- Guest checkout only (no customer accounts) — simplicity + COPPA concerns
- SMS notifications deferred to v2 — email + dashboard sufficient for launch
- Flat-rate shipping
- Auto charity counter from completed orders
- Clean minimal design — products front and center
- Interactive mode, standard granularity, parallel execution, all workflow agents enabled, inherit model profile
